# PartPickr Lab

This is a standalone version of the PartPickr Lab application built with plain HTML, CSS, and JavaScript. It does not require Node.js, React, or any build tools.

## How to use:
1. Open `index.html` in any web browser.
2. When prompted, enter your **Gemini API Key**.
3. Choose between "Full Build" or "Upgrade".
4. Enter your requirements and click **EXECUTE**.

## Files:
- `index.html`: The structure of the app.
- `style.css`: The "Retro-Cinematic" styling and glassmorphism.
- `script.js`: The logic for API calls and UI rendering.

## Hosting:
You can host this easily on:
- **GitHub Pages**: Just upload these files to a repository.
- **Netlify/Vercel**: Drag and drop the folder.
- **Local Server**: Use VS Code Live Server or just open the file directly.
